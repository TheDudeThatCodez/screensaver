# CLI Screensaver Rotator

Rotates various terminal screensaver-style programs every 12 hours.

## Included Tools

- cmatrix
- hollywood
- pipes.sh
- asciiquarium
- sl
- bb
- cacafire
- aafire
- cbonsai
- nyancat

## Setup

```bash
git clone https://github.com/yourusername/cli-screensaver-rotator.git
cd cli-screensaver-rotator
chmod +x install.sh
./install.sh
